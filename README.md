# Mtafiti Soko Limited  
**A modern, scalable, and culturally-rooted academic marketplace + AI training SaaS platform**  
designed for **Kenyan university students** and **vetted expert Mtafitis** in **International Relations, Psychology, History, and Business Studies**.

This is **not just a freelance site** — it’s a **complete ecosystem** combining:
- **Academic support** (term papers, theses, data analysis)  
- **AI Kazi training** (non-coding AI tools for research & productivity)  
- **Community-driven trust** via campus ambassadors and local payments (M-Pesa)

> **Execution Model**: Fiverr/Upwork-inspired workflow — **Request Dump → Instant Quote → Secure Pay → Track → Deliver**  
> **But 100% Kenyan**: M-Pesa, Swahili UI, campus culture, local academic standards

---

## Business Model: B2C + C2B + B2B Hybrid

| Model | Description |
|------|-------------|
| **B2C** | Students **dump** project → get **instant quote** → pay via M-Pesa → track in real-time |
| **C2B** | Mtafitis (experts) **claim** gigs from dynamic queue → deliver via secure workspace → get **weekly payouts** |
| **B2B** | Universities license **white-label portals** for cohort support + AI Kazi workshops |
